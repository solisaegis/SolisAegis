## Summary

**Latency as Intelligence** argues that interpretive latency can function as intelligence when the interval before action preserves comparison, context, reversibility, and correction under uncertainty.

The document distinguishes speed from judgment and clarifies that latency is not mere slowness, not a command to delay, and not a design rule for artificial intelligence. It focuses on interpretive tempo: the structural pacing by which perception, interpretation, response, and consequence are coordinated before action collapses a field of uncertainty.

---

## Core Claim

> Interpretive latency can function as intelligence when the interval before action preserves comparison, context, reversibility, and correction under uncertainty.

This document does not claim that slower systems are always wiser, that delay always improves judgment, that speed is inherently unintelligent, that hesitation guarantees safety, that latency produces morality, or that artificial intelligence systems should be forced to delay outputs.
