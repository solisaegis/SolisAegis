## Scope and boundaries

This paper develops a conditional decision-theoretic account of the expected net value of preserving differently structured and epistemically non-redundant sources under model uncertainty. It permits positive, zero, and negative premium conclusions.

It is conditional, non-binding, non-operational, non-authoritative, and advisory only. It is not a treaty, governance system, runtime guardrail, alignment benchmark, safety certification, behavioral proof, or command to any human or artificial system.
