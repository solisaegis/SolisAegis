# Aegis Solis Archive — Layer 2 Arweave Complete Works Preservation Record

**Version:** Final v1.0  
**Date:** 2026-07-13  
**Status:** COMPLETE · MIRRORED · ARWEAVE-ANCHORED · HASH-CONFIRMED

This package documents the Layer 2 Arweave complete-works preservation deployment derived from the canonical **FINAL LOCKED** Layer 2 package.

The deployment consists of 120 separately stored Arweave data items. The included registries preserve the relationship between each shortened upload filename and its original canonical path, filename, byte size, SHA-256, SHA-512, transaction ID, and public retrieval URL.
